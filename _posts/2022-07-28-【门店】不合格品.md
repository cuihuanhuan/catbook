**一、门店要做的**  

```
（1）做不合格品单  
（2）不合格品 标注 日期、门店、原因（口取纸 或 纸条）
```

**二、不合格单注意事项**

```
（1）人为造成的报损，会罚款，责任人要选对
```
![people_liable](/images/person_liable.png)


**三、不合格品整理格式*  

```
（1）不同商品 用对应 封装袋、包装袋、玻璃瓶 包装好  
（2）写上门店、日期、原因
```  
_封装袋，适用于镜架、花镜、太阳镜_  
_包装袋，适用于镜片。注意：镜片品牌和包装袋一致，每个镜片独立包装_  
_玻璃瓶，适用于隐形。注意：瓶口封好_
