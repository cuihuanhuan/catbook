**一、积分类型**
```
（1）转介绍积分
（2）补积分
（3）减积分
（4）镜小二积分转移（镜小二积分 转移到鹏升系统，积分一减一增）
（5）积分转移（鹏升系统里 积分 互相转移，积分一减一增）
```


**二、镜小二积分转移 注意**  
```
1.如果是镜小二积分转移，一定要写上 "镜小二积分转移",否则镜小二没做减积分，后续可能会出现 积分重复赠送的情况

  例，官湖 积分转移，每次都不写镜小二积分转移，我都是自己搜到镜小二，然后减镜小二积分
  
2.店长根据自己理解，做合适的调整，避免出现积分重复赠送情况
```



**三、书写格式**
```
（1）张三 13012345678 介绍 小王 15845671236 配镜 580
（2）李四 13012343333 介绍 小赵配镜 15845677894 积分 280
（3）小红 12234569871 积分 -500
（4）小明 17734567890 积分 +280
（5）赵七 13345678901 积分 -350，赵七妹妹 13345679999 积分 +350  
（6）镜小二积分转移，小美 13356781234 积分 +1200
```
